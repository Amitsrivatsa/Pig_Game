Abstract


In this project we have taken "goShopping_WebClicks.dat" and "goShopping_Iplookup.txt" as the datasets on which we have to perform some analysis.Two kinds of analysis is done based on Real time data as well as batch data Hive Quey Language, Map-reduce, Spark-core and Spark-sql is used for analysis of batch data. Spark-kafka-streaming is used for Real time analysis of data. The visualization of Map-Reduce data is done through Tableau.
